Advanced

After downloading, follow these steps:

  • 1 Install Python from python.org if not already installed. Check "Add to PATH" during installation on Windows.
  • 2 Open a terminal and install dependencies:
    pip install pyserial requests
  • 3 Open the downloaded file in a text editor. Fill in your sensor API keys at the top. You can find them in FCHAN → your zone → click your sensor.
  • 4 Plug in your Arduino, then run:
    python3 fchan_usb_reader.py (Linux/Mac)
    python fchan_usb_reader.py (Windows)
  • 5 Open FCHAN in your browser — your sensor readings will appear live.
