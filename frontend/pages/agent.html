Arduino Code Format

For best results, program your Arduino to print in this format. Any format with a colon separator works.

// Recommended Arduino Serial format
Serial.print("TEMP:");
Serial.println(temperature);

Serial.print("HUM:");
Serial.println(humidity);

Serial.print("SOIL:");
Serial.println(soilMoisture);

Serial.print("LIGHT:");
Serial.println(lightIntensity);

// Or any label: value format
// e.g. "Temperature: 28.5"
// e.g. "water sensor value :266"
